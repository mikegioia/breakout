Breakout Game – Lesson 01

Task: download the first version of the game file to your USB drive

What To Do

  1. Open the game file in a new tab
    Click this link to open the game in a new tab
  2. Right-click the game screen and click "View page source"
  3. Highlight all of the code and copy it to your clipboard
    Command + c will copy on Mac computers
  4. Open your text editor and paste the code into the new file
  5. Save the file to your USB drive
    Make sure to plug the USB drive in if you haven't yet
